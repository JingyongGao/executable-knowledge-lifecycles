First public evidence freeze for **Executable Knowledge Lifecycles as Dynamical
Control in Agent Systems**.

- strict natural-language-to-causal-IR compilation with sealed governance fields;
- 18 manually authored semantic cases and 390 explicitly labeled decode events;
- separate SemanticHash and EnvelopeHash conformance contracts;
- correct, wrong, partial, stale, and fault-activated stale execution conditions;
- paired-seed lambda and confidence sensitivity scans;
- a 62-test invariant suite;
- a TeX-built, visually audited three-page short paper;
- deterministic evidence ZIP, release manifest, and SHA-256 checksums.

## Interpretation boundary

The release uses synthetic linear environments and a small handcrafted semantic
benchmark. It demonstrates lifecycle control behavior, not causal discovery or
production readiness in any named domain. See `LIMITATIONS.md`.
